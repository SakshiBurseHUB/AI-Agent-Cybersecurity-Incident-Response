# Source Code

This directory contains the core implementation of the **AI Agent for Cybersecurity Incident Response**.

## Purpose

The `src/` folder contains all application logic, including log collection, threat detection, AI analysis, response recommendations, and application startup.

## Files

| File            | Purpose                                 |
| --------------- | --------------------------------------- |
| `main.py`       | Entry point of the application          |
| `analyzer.py`   | AI-based threat analysis                |
| `classifier.py` | Classifies cyber attack types           |
| `playbooks.py`  | Response recommendations and playbooks  |
| `logger.py`     | Logging utilities                       |
| `__init__.py`   | Marks the directory as a Python package |

## Development Guidelines
